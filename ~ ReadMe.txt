Data pertaining to Maskalenko et al. 2024, 
"The FcyRIIIA (CD16) L48-H/R polymorphism enhances natural killer cell-mediated antibody-dependent cellular cytotoxicity"
The directory "Figures" contains assembled figures for this manuscript.
The directory "Figure Data" contains data values used to assemble the figures from the manuscript.
The directory "Imaging Data" contains imaging data from the project.
Due to size constraints, only one experiment was included for Figures 2 and 3 (1/6 replicates, 1/3 replicates).
All imaging data from Figures 4 and 5 are included.
